# Gerando os pdfs

Execute o comando `make pdf`. Os arquivos pdfs serão gerados no diretório `target/pdfs`.

Este comando irá baixar e executar os programas [pandoc](https://pandoc.org/)
e [tectonic](http://tectonic-typesetting.github.io/).
